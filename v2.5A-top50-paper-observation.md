# v2.5A Top50 Paper Observation

> 目标：测试 MinSwing v3 在 OKX Top50 USDT 市场的泛用性
> 不是 v2.5 hybrid OOS，不是 v2.6，不升级策略

## 阶段划分

```text
v2.5A-1: Top50 Universe + Data Preflight
v2.5A-2: Top50 Paper Observation (preflight 通过后)
```

## 核心规则

```text
- OKX SPOT USDT Top50，不混 swap
- 不优化参数，新币用 default config
- current_exit = MinSwing v3 default
- hybrid_exit 不启用
- 不允许把 Top50 结果用于策略晋级
- 所有结果写 research.duckdb
```

## 停止条件

```text
- 连续 3 个 heartbeat failed → auto stop, session_status=failed
- latest_bar_delay > 2 根 5m → 停止 accepted trades，只记录 signals
- DB 写入失败 → 立即停止
- disk_free < 5GB → 立即停止
```

## 输出

```text
config/universe/okx_top50_YYYYMMDD.yml
reports/v2_5A_top50_paper_observation/heartbeat/YYYYMMDD_HH.md
data/research/paper_sessions/ (per session artifacts)
research.duckdb (paper_sessions + signals)
```

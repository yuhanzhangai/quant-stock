# v2.5 True OOS Hybrid Shadow Paper

> 前置条件：v2.4.1 decision amendment 完成
> 目标：用 v2.4 之后的新数据验证 hybrid_exit 是否真正有效
> 这是唯一能回答 A2 质疑的方式

## 核心原则
```text
[ ] current_exit 继续是 Production default
[ ] hybrid_exit 只 shadow，不实际执行
[ ] 不重复开仓
[ ] 不改 entry
[ ] 不调 hybrid 参数
[ ] 不根据中途表现修改规则
[ ] 只使用 v2.4 之后产生的新 entry
```

## 数据独立性要求（硬性）
```text
entry_id 不得出现在 v2.3R / v2.4 common_entries 中
overlap_with_v2_4_entries = 0
overlap_with_v2_3r_entries = 0

v2.5 报告必须输出:
  new_entries_count
  overlap_with_v2_4_entries
  overlap_with_v2_3r_entries
```

## 最低完成条件
```text
[ ] 至少 30 天 shadow observation OR 100 new entries
AND
[ ] differential_exit_count >= 30

如果 30 天后 differential_exit_count < 30:
  结论只能是 inconclusive_continue_shadow
```

## 新增 Gates（v2.4 教训）
```text
concentration_gate: top_5_trade_pnl_share <= 60% (v2.4 was 78.8%)
  60-70% = WARNING
  >70% = FAIL

drawdown_gate: hybrid_max_dd <= current_max_dd * 1.05
  hybrid 允许轻微波动但不能明显恶化

fee_gross_gate: hybrid fee/gross <= current + 10%
  任何窗口 fee/gross > 100% = WARNING

differential_trade_gate: differential_exit_count >= 30
  不够则 inconclusive
```


## 结论四选一
```text
1. continue_shadow — 数据不够，继续观察
2. promote_to_formal_exit_mode_experiment — OOS 支持 hybrid → v2.6
3. remain_research_exit_mode — 有优势但不稳定
4. reject_hybrid_exit — OOS 不支持 v2.4 发现
```

## 不允许
```text
promote_to_production
promote_to_default

```
